# 📝 Todo App (Dynamic Website)

A clean, responsive, and dynamic Todo application that helps users manage their daily tasks efficiently. This project features a modern user interface with an integrated light/dark theme toggle for a personalized user experience.

## 🚀 Features
* *Task Management:* Seamlessly add and save tasks.
* *Theme Toggle:* Switch between light mode and dark mode using a simple UI button.
* *Responsive Design:* Fully optimized for both desktop and mobile viewing.
* *Persistent Storage:* (Optional: Add this if you used LocalStorage to save tasks upon refreshing).

## 🛠️ Tech Stack
* *HTML5:* Semantic structure for web content.
* *CSS3:* Custom styling and smooth animations for theme transitions.
* *JavaScript (ES6):* Dynamic DOM manipulation for task actions and state management.
